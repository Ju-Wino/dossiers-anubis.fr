Guide


ANUBIS ne se contente pas de contenir les anomalies. Elle les étudie et, si possible, les exploite.


PROTOCOLES DE SÉCURITÉ


- Tout contact avec une anomalie doit être supervisé par un agent de la sécurité.

- Les expériences sur les anomalies doivent être approuvées par le Conseil.

- Si une anomalie devient incontrôlable, activation de la Protocole "Passeur de l'Ombre"

(extermination, scellage, ou utilisation d'une autre anomalie contre elle).

- En cas de brèche de sécurité, l'information doit être effacée de toutes les bases de données externes.


INTERDICTIONS ABSOLUES


- Il est formellement interdit de tenter d'exploiter une anomalie sans autorisation du Conseil.

- Aucun membre ne doit entretenir de relation personnelle avec une anomalie.

- Toute interaction prolongée avec une anomalie psychiquement influente doit être suivie d'un contrôle mental obligatoire.
